# Learning Objectives Generator

Generate learning objectives for coding tasks using OpenAI.

## Setup

1. Install dependencies:
```bash
   npm install
```

2. Create `.env` file and add your OpenAI API key:
```
   OPENAI_API_KEY=your_api_key_here
   PORT=3000
```

3. Start the server:
```bash
   npm start
```

4. Open http://localhost:3000 in your browser

## Usage

1. Enter a coding task (e.g., "build a login form")
2. Specify the technology (e.g., "React")
3. Click "Generate Learning Objectives"
4. Review the skills you'll learn!

## Tech Stack

- **Frontend:** Pure HTML, CSS, JavaScript
- **Backend:** Node.js + Express
- **AI:** OpenAI API (GPT-4)
